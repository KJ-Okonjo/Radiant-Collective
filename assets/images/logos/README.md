# Logos Directory

Add partner and client logos here.

## Recommended Formats
- PNG (transparent background preferred)
- SVG (vector format for scalability)
- JPG (if necessary)

## Recommended Dimensions
- Width: 180-250px
- Height: 80-120px
- Maintain aspect ratio

## Usage
These logos are used in:
- Home page ticker section
- Partner showcase areas
- Footer (if needed)
