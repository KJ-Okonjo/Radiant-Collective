# Radiant Collective

Leadership Mastermind Experience for Women in Africa

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Modern CSS

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Deploy

Automatically deploys to Vercel on push to main branch.
